# CCNA Practice Labs and Resources 📡

This repository contains my **CCNA practice labs, Packet Tracer files, notes, and related study materials** gathered throughout my networking learning journey.

The goal of this repository is to document my practical work, organize useful resources, and share them publicly for other networking learners.

It serves as a collection of hands-on practice materials for strengthening networking fundamentals and improving real-world CCNA skills, including routing, switching, subnetting, and network configuration.

---

## 📚 What’s Included

- 🖧 Network topology practice (Packet Tracer)
- 🧪 Cisco Packet Tracer files (.pkt / .pka)
- 📊 Subnetting & IP addressing practice
- 🔁 Routing protocols (RIP, OSPF, EIGRP)
- 🔒 NAT & ACL configurations
- 📄 Study notes and reference materials
- 💻 Lab-based configuration exercises
